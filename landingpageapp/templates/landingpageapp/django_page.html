This website was developed using the Django web framework.

The following features and capabilities of Django were leveraged to create a robust and efficient website:

  • Django Class-Based Views (CBV): The website's views were structured using Django's Class-Based Views, providing a modular and reusable approach to handle HTTP requests and serving content.
  • Django Authentication: The website incorporates user registration, login, and authentication mechanisms, courtesy of Django's built-in authentication system.
  • Django Templates: The website's user interface was crafted using Django's templating engine, offering a dynamic and consistent user experience.
  • Django Forms: This provided a systematic way to handle user input, ensuring data validation and security.
  • Django ORM (Object-Relational Mapping): This allowed for seamless interactions with the database, enabling efficient data storage and retrieval.
  • Django Static & Media Files Handling: Efficiently managed the delivery and storage of static assets like CSS, JavaScript, and images, as well as user-uploaded media files.
  • Django Path Converters: For creating Dynamic URLs within the site, Django's path converters were harnessed, allowing for greater flexibility in URL patterns and content delivery.
